³ Introduction to ACiD View  º
ÀÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍ¼

            We at  ACiD Productions, LLC  proudly  present  to  you the
        first  Windows-based  viewer  for  the  artscene.  This  highly
        innovative program is capable  of  displaying a wide variety of
        formats such as ANSI, ASCII, ADF, BIN, GIF, JPEG, PNG, RIPscrip
        & XBIN with additional supported  formats  underway.  ACiD View
        for Windows utilizes its own internal ANSI routines  making  it
        capable of viewing ANSI without  fussing with the configuration
        of an internal device  driver.  We  hope  that  this version of
        ACiD View provides you with  the  countless  hours  of pleasure
        that we've sought after with its creation.

³ What ACiD View Displays  º
ÀÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍ¼

     ş  ACiD View will display any traditional text-mode format, not just
        ANSI, in several different sizes.  Where as previous DOS versions
        had various "VGA modes" you could use to scale down the size of
        an ANSI, ACiD View utilizes your native display resolution and
        offers several different font sizes to accomplish the same effect.
        Supported sizes are: Normal (standard 8x16 font), Half-size (4x8),
        Quarter-size (2x4) and Eighth-size (1x2).

     ş  ACiD View shows GIF, JPEG and PNG graphics files.

     ş  Support for other popular BBS image formats is in development.

³ Using ACiD View with Windows Explorer  º
ÀÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍ¼

 When ACiD View is selected as the default viewer for file type such as
 ANSI and ASCII, several options are available to the user from the
 Windows file Explorer.

 - Slide Show can be launched from Explorer by selecting any number of
   images supported by ACiD View and selecting "Slideshow" from the
   right-button context menu.

 - Drag and Drop Loading.  Images can be dragged from Explorer right
   in to ACiD View for display.


ÚÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄ·
³ ACiD View Slide Show Options    º
ÀÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍ¼

 Within the ACiD View GUI, there are several preference settings which can
 be configured.

 Full Screen Resolution - Four (4) resolutions are available to choose from
 if Full Screen mode is enabled; 640x400, 640x480, 1024x768 and the Desktop
 resolution.

 Slide Show:

 Files to be displayed in the Slide Show may be queued one of two ways,
 either by a standard ASCII text file list of files ("Text File") or by
 displaying the entire contents of a chosen Directory, RAR or ZIP file, the
 default being the current directory.

 Start Full Screen - This will start the Slide Show in Full Screen with the
 resolution specified above.

 Random - This will display the files queued in a random order.

 Cycle - This will endlessly cycle through all the files in the queue until
 the F8 or Escape key is pressed.

 Delay - This is the amount of time to pause before showing the next image in
 the queue.  The default value is 2000 milliseconds.

 Other options such as File Mask, Font Size, Font Set, and Color are based on
 the current settings.


ÚÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄÄ·
³ ACiD View Command Line Viewing  º
ÀÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍÍ¼

 ACiD View now supports a full set of command line switches which
 are useful to execute a Slide Show.

 At present, you can specify a specific file you would like to view
 from the command line using the following syntax:

  ACiDview.exe [drive:][path][filename]

  i.e.: ACiDview.exe R:\ACiD\Artpacks\Best_of\best.1994\SO-EVEND.ANS

ACiD View for Windows Command Line Options:

 ACiDview.exe /X /R:# /F:# /C:# /D:# /S:[0|1] /Y:[0|1] /A:[0|1]
(continued)   /M:mask(s) @Inputname(s)

 /X - required to execute Slide Show.

 /R (Font Size)  /F (Font Set)     /C (Color)         /D (Display)
  0 - Normal      0 - Normal        0 - Normal         0 - 640x400
  1 - MCGA        1 - 50 Lines      1 - Greyscale      1 - 640x480
  2 - VGA         2 - Amiga         2 - ASCII          2 - 1024x768
  3 - SVGA        3 - AcidView                         3 - Desktop

 /S - If 0, start display in a WINDOWED mode.
      If 1, start FULL SCREEN using the display resolution specified
      with the /D switch (or use the default value in the registry).
 /A - If 0, then only show the files in the queue once.
      If 1, then continuously repeat the Slide Show AGAIN once all
      the files in queue have been shown.
 /Y - If 0, display files in the order they are queued.  
      If 1, display queued files at RANDOM.
 /M - MASK(s) off semicolon delimited extension(s) listed in
      quotes, e.g. "/M:*.ans;*.jpg" to be displayed.

 @Inputname(s) - Space delimited file list that can include text file
 lists (ending in .TXT), directories (ending with \), compressed files
 (ending in .ZIP or .RAR), or supported graphics files.

     -=- Default values are loaded from the registry -=-

Examples:

 ACiDview.exe /X acid-81a.rar acid-81b.rar

     This will start a SLIDE SHOW and display all supported images
     within acid-81a.rar and acid-81b.rar

 ACiDview.exe /X /R:0 /F:1 /C:0 /D:0 /S:1 /Y:1 /A:1 /M:*.ans
(continued)   acdu0194.zip acdu0294.zip favfiles.txt JD-DSOC2.ANS testdir\

     A complex heap of options enabled, this example will begin a
     Slide Show using NORMAL attributes for the font size and color
     with a 50 LINE mode font set and FULL SCREEN display resolution
     of 640x480.  ACiD View will proceed to display only the ANSI files
     ending in .ANS within the ZIPs, those listed in the text file
     favfiles.txt which end in .ANS, the local file JD-DSOC2.ANS,
     and finally all the .ANS files in the relative local directory
     testdir\, all in RANDOM order, repeating the Slide Show infinitely
     over and over until the F8 or Escape key is pressed.
