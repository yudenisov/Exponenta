Exponenta
=========

Пакет для удалённого взлома компьютера и управления им. Сочетает в себе множество функций, поэтому автор не может отнести его к определённому классу зловредов. Наиболее близкий класс в категории зловредов — стайлеры (Stealer)

Для своей работы данный пакет требует дополнительные пакеты и программное обеспечение. Ссылки можно найти в документации.

Кроме того, при своей работе пакет требует идентификации хакера путём размещения в скомпилированных модулях пакета закодированного base64 идентификатора хакера и задачи. Это сделано с целью предотвращения криминального распространения программы и привязки процессов ботнета к единому идентификатору.

Заявленные функции
==================

* Возможноа скрытая установка методами социальной инженирии.
* При своей работе требует права локального администратора.
* Позволяет скрытно устанавливать пакеты на заражённый компьютер.
* Позволяет запускать файлы на удалённом компьютере. Результат пернедаётся «Хозяину» по FTP протоколу

Системные требования
====================

* Компьютер с установленной Microsoft Windows 7 SP1 — Windows 10 Pro или Microsoft Windows Server 2008 R2 — 2016.
* Учётная запись локального администратора компьютера или администратора домена.
* Установленные в системе пакеты PowerShell 2.0, Console FTP client, systeminfo.exe. Эти пакеты изначально присутствуют в указанных системах, они просто не должны быть удалены администратором.

Необходимые дополнительные пакеты для работы
============================================
* Менеджер пакетов chocolatey.
* Модифицированная версия программы Elevation.
* Программа Hidden Start (зарегистрированная версия).
* Установленный из Chocolatey пакет sysinternals.
* Установленный из Chocolatey пакет curl.
* Установленный их Chocolatey пакет wget.

Программа Exponenta устанавливает их сразу же после своей инсталляции.
