Установка пакета
==============

- Установите зависимости:

```sudo apt-get install -y bash sed coreutils diffutils M4 make perl autoconf autotool```

- Скачайте данный пакет с репозитория и перейдите в эту папку
- Подайте команды:

```autoreconf -iv```

```./configure```

```make```

```sudo make install```

``` make clean```

Удаление пакета
=============

- Перейдите в этот каталог
- Подайте команду:

```sudo make uninstall```
